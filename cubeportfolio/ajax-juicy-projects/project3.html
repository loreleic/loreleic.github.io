Content Management System
Python-based platform that powers custom sites

Overview: Content management system built for internal users in order to meet the unique features of Best Of and Storytellers. Written in Python and hosted on S3.

Features: Customized feature set for both sites that includes account management, search, drag and drop, built-in crop tool, Google maps API integration, Brightcove video integration, WYSIWYG, JSON feed export to update front-end node.js sites.
