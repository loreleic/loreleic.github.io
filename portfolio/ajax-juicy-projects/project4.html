Lead Acquisition Tool
Captures emails and user data at events.

Overview: Responsive mobile app used at sponsored local events to capture email addresses and user data for lead generation efforts. In addition, a secondary tool was created that allows marketers to batch-upload data post-event.

Features: Ability to select market that triggers custom branding, create events and lead pages, offline sync, integration with Formstack, back-end processing that sends data to Exact Target, account management, bulk import support for CSV files, and lead segementation.

Metrics: The app has been used thus far by 14 markets, has approximately 14,000 submissions from over 150 events.
